## Purpose

This template provides a comprehensive framework for planning marketing campaigns using the RACE methodology (Plan → Reach → Act → Convert → Engage). It ensures strategic alignment, clear objectives, and coordinated execution across all campaign phases.

## Agent Notes

### For Campaign Strategist
- Ensure all RACE phases have clear objectives and success metrics
- Validate cross-functional alignment before launch
- Build in 10-20% contingency budget for opportunistic spending
- Reference best practices from marketing-campaign-best-practices.md

### For Marketing Analyst
- Define measurement approach early (attribution model, tools, privacy compliance)
- Set up tracking infrastructure before launch
- Create automated dashboards for real-time monitoring
- Plan for incrementality testing to prove channel value

### For Channel Specialist
- Coordinate channel mix across RACE phases
- Ensure consistent UTM tagging across all channels
- Build channel-specific optimization playbooks
- Monitor cross-channel attribution and assist paths
